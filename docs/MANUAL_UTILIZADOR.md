# Manual do Utilizador — Find

## Instalação

1. Instala a aplicação no dispositivo Android (ou executa no emulador)
2. Garante que a localização está ativa
3. Aceita a permissão de localização quando solicitada

## Ecrã principal — Mapa

- Ao abrir, o mapa centra-se na tua localização atual
- Marcadores verdes indicam bancos aprovados
- Toca num marcador para ver detalhes

**Barra superior:**
- 🔍 Pesquisar bancos por características
- 🔑 Entrar (se não autenticado)
- ⚙️ Painel admin (só administradores)
- 🚪 Sair (se autenticado)

**Botões:**
- 📍 (canto inferior esquerdo) — voltar à localização atual
- ➕ (canto inferior direito) — reportar novo banco (requer login)

## Sem conta

Podes usar a app **sem login** para:
- Ver o mapa com bancos aprovados
- Pesquisar bancos
- Ver detalhes de cada banco

## Criar conta / Entrar

1. Toca no ícone de login
2. Introduz email e password
3. Ou cria conta nova em "Criar conta"

## Reportar um banco

1. Entra na tua conta
2. Toca no botão ➕
3. A app usa a localização GPS atual
4. Seleciona tipo (madeira, cimento, etc.)
5. Indica cor e largura
6. Confirma — o banco fica pendente até aprovação

## Avaliar um banco

1. Abre o detalhe de um banco aprovado
2. Seleciona 1 a 5 estrelas
3. Toca em "Guardar avaliação"

## Pesquisar

1. Toca em 🔍 no mapa
2. Filtra por cor, largura e/ou tipo
3. Toca em "Pesquisar"
4. Seleciona um resultado para ver detalhes

## Administrador

Conta demo: `admin@find.pt` / `admin123`

1. Entra com conta de administrador
2. Toca no ícone de admin
3. Aprova ou rejeita bancos reportados
